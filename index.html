Acerca de esta herramienta

Este mapa interactivo permite explorar el uso de suelo silvoagropecuario en las regiones de Chile, mostrando la superficie estimada en hectáreas para distintos rubros productivos.

Rubros disponibles: Agrícola, Forestal, Ganadero, Frutícola, Apícola y Acuícola.

⚠️ Los datos presentados son simulados para efectos de demostración (MVP). Para datos oficiales consultar el ODEPA o el INE.
